# Type Speed Game
A typing speed game built with **HTML, CSS, and JavaScript only**.

##  How it works
- A random word appears on the screen. -
- You must type it correctly within **2, 3, or 5 seconds**, depending on the chosen difficulty.
- The game offers **three difficulty levels**.
-  At the end, your score is calculated out of **30 points**.
- ⚠️ If you fail even once, the game ends immediately and you must start over.

  
 ##  Features
 - Pure front‑end implementation (no frameworks). -
 -  Random word generator. - Countdown timer based on difficulty. -
 -   Instant game over on a single mistake. -
 -    Score tracking system.
